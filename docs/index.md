# Resume Studio

**One master CV across languages. Targeted PDF and Word exports for each audience.**

Resume Studio is a small, self-hosted resume manager built for consultants who
keep one CV up to date in several languages and need to produce different
versions of it for different clients, tenders, or roles.

You edit your full master CV once — in any language you've added — and the app
shows every translatable field side-by-side in two languages so translations
never drift out of sync. When you need to send something out, you build a
**Resume View**: pick the sections that matter to this audience, exclude the
items that don't, add a custom intro, and export to PDF or `.docx`.

It runs as a local desktop app on Windows, macOS, and Linux — your data stays
on your own machine. A multi-machine sync option uses a folder in Google
Drive, Dropbox, or OneDrive so the same set of CVs follows you between
computers.

## Who it's for

- Independent consultants and contractors who pitch in several languages.
- Anyone tired of keeping three half-translated `.docx` files in sync.
- People who'd rather own the data file than upload their CV into a SaaS.

## What makes it different

- **Edit in two languages at once.** Every field shows both languages on
  screen — no toggling, no "wait, did I update the English version?".
- **Resume Views, not duplicate documents.** Build a Board-level CV, a deeply
  technical CV, and a one-pager from the *same* underlying data. Change a
  project once, all three export updated.
- **Your data stays yours.** No account, no cloud database. The app runs on
  your computer; an optional backup folder uses a cloud-sync service you
  already trust.
- **Privacy-respecting translation assist.** Optional draft translations run
  through a self-hosted [LibreTranslate](https://libretranslate.com/) instance
  (or a key-based provider you choose). CV text only ever travels between your
  app and the provider you configured — there's no Resume Studio backend.
